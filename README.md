# AI Intern - Task 1

Този проект има за цел да автоматизира процеса по извличане на информация от протоколи на срещи и съхраняването им в облачна база данни.

## Функционалности
- **Автоматично обхождане:** Скриптът претърсва директория `data/` за проекти и техните срещи.
- **Обработка на документи:** Четене на съдържанието на `.docx` файлове с помощта на `python-docx`.
- **Интеграция с облак:** Съхранение на метаданни и транскрипции в **Supabase** (PostgreSQL).
- **Сигурност:** Използване на `.env` за управление на API ключове.

## Технологичен стек
- **Език:** Python 3.12
- **База данни:** Supabase
- **Библиотеки:** `supabase-py`, `python-dotenv`, `python-docx`

## Инструкции за инсталация

1. **Клонирайте хранилището:**
   git bash
   git clone https://github.com/entodorov/ai-intern-task1

3. **Конфигурирайте променливите на средата:**
Създайте файл `.env` в основната папка и добавете:

SUPABASE_URL=your_project_url
SUPABASE_KEY=your_anon_key

## Начин на работа

1. Подредете Word файловете в папка `data/ProjectName/`.
2. Стартирайте скрипта за качване:
    bash
    python ingest.py
3. Проверете качените данни:
    bash
    python query.py

## База данни (Schema)
- **Таблица `meetings`**: Пази суровия текст и информация за срещата.
- **Таблица `notes`**: Предвидена за бъдеща обработка с AI.
